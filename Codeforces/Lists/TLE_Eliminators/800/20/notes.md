

- Find any permutation b of length n such that a1+b1≤a2+b2≤a3+b3≤…≤an+bn.

- a0 a1 a2 a3 ... an
- b0 b1 b2 b3 ... bn
- a0+b0 a1+b1 a2+b2 a3+b3 ... an+bn
- a0+b0 ≤ a1+b1 ≤ a2+b2 ≤ a3+b3 ≤ ... ≤ an+bn

- to find b.

# Solution:
- make a0+b0 = a1+b1 = a2+b2 = ... = an+bn
- the whole logic is this only.
- a0+b0 = a1+b1 = a2+b2 = ... = an+bn = k
- k will be n+1
